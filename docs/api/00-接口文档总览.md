## 4 数据流向

```
请求流 (Request):
  Frontend → Gin Router → Auth MW → Casbin MW → Controller → Service → DAO → MySQL
                                                                      → Redis

响应流 (Response):
  MySQL/Redis → DAO → Service → Controller → Response.JSON → Frontend

跨模块调用 (当前为同进程):
  BorrowService ──GetForUpdate(tx)──► EquipmentDAO   (同一 DB 事务内)
  BorrowService ──UpdateWithTx(tx)──► EquipmentDAO   (同一 DB 事务内)
  AuthService   ──FindByUsername──►  UserDAO         (独立查询)
```
