# 01 — 通用数据结构

## 1 统一响应格式

### 1.1 Response

所有 API 返回的 JSON 最外层结构：

```go
// pkg/response/response.go
type Response struct {
    Code int         `json:"code"` // 业务错误码，0 = 成功
    Msg  string      `json:"msg"`  // 提示信息
    Data interface{} `json:"data"` // 业务数据，错误时为 null
}
```

**示例：**
```json
// 成功
{"code": 0, "msg": "success", "data": {"id": 1, "username": "zhangsan"}}

// 失败
{"code": 2001, "msg": "用户名或密码错误", "data": null}
```

### 1.2 PageResult

分页响应统一使用此结构嵌入 `data`：

```go
// pkg/response/response.go
type PageData struct {
    Total    int64       `json:"total"`
    Page     int         `json:"page"`
    PageSize int         `json:"page_size"`
    List     interface{} `json:"list"`
}
```

**示例：**
```json
{
    "code": 0,
    "msg": "success",
    "data": {
        "total": 50,
        "page": 1,
        "page_size": 10,
        "list": [ ... ]
    }
}
```

## 2 错误码全集

```go
// pkg/errcode/errcode.go
const (
    // ── 成功 ──
    Success = 0

    // ── 通用 (1xxx) ──
    ErrInvalidParam = 1001 // 请求参数错误
    ErrNotFound     = 1002 // 资源不存在
    ErrConflict     = 1003 // 资源冲突

    // ── 认证与权限 (2xxx) ──
    ErrAuthFailed       = 2001 // 用户名或密码错误
    ErrAccountDisabled  = 2002 // 账号已被禁用
    ErrTokenMissing     = 2003 // 未提供认证 Token
    ErrTokenInvalid     = 2004 // Token 无效或已过期
    ErrPermissionDenied = 2005 // 权限不足

    // ── 业务 (3xxx) ──
    ErrUserExists          = 3001 // 用户名已存在
    ErrUserNotFound        = 3002 // 用户不存在
    ErrEquipmentNotFound   = 3003 // 设备不存在
    ErrStockInsufficient   = 3004 // 设备库存不足
    ErrDuplicateBorrow     = 3005 // 已有该设备的未归还借阅
    ErrBorrowApproveFailed = 3006 // 审批失败（库存不足）
    ErrBorrowStatusInvalid = 3007 // 工单状态异常

    // ── 系统 (5xxx) ──
    ErrInternalServer = 5000 // 系统内部错误
)
```

**HTTP 状态码约定：**

| HTTP Status | 使用场景 |
|-------------|---------|
| 200 | 成功查询、更新、删除 |
| 201 | 成功创建 |
| 400 | 参数校验失败 (code: 1001) |
| 401 | 认证失败 (code: 2001/2003/2004) |
| 403 | 权限不足 (code: 2005) |
| 404 | 资源不存在 (code: 1002) |
| 409 | 资源冲突 (code: 1003/3001) |
| 500 | 系统内部错误 (code: 5000) |

## 3 分页请求规范

所有列表类接口统一使用以下 Query 参数：

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| page | int | 否 | 1 | 页码，从 1 开始 |
| page_size | int | 否 | 10 | 每页条数，最大 100（超出自动截断） |

**Go 绑定结构体：**

```go
type PageReq struct {
    Page     int `form:"page"      binding:"omitempty,min=1"`
    PageSize int `form:"page_size" binding:"omitempty,min=1,max=100"`
}
```

## 6 请求/响应结构体汇总

### 6.1 认证模块

```go
// ── 请求 ──
type LoginReq struct {
    Username string `json:"username" binding:"required,min=3,max=32"`
    Password string `json:"password" binding:"required,min=6,max=64"`
}

type RefreshReq struct {
    Token string `json:"token" binding:"required"`
}

// ── 响应 ──
type LoginResp struct {
    Token     string `json:"token"`
    ExpiresIn int64  `json:"expires_in"`
}
```

### 6.2 用户模块

```go
// ── 请求 ──
type CreateUserReq struct {
    Username string `json:"username"  binding:"required,min=3,max=32,alphanum"`
    Password string `json:"password"  binding:"required,min=6,max=64"`
    RealName string `json:"real_name" binding:"required,min=2,max=32"`
    Email    string `json:"email"     binding:"omitempty,email,max=64"`
    Phone    string `json:"phone"     binding:"omitempty,numeric,max=16"`
    RoleID   uint   `json:"role_id"   binding:"required,gt=0"`
}

type UpdateUserReq struct {
    RealName string `json:"real_name" binding:"omitempty,min=2,max=32"`
    Email    string `json:"email"     binding:"omitempty,email,max=64"`
    Phone    string `json:"phone"     binding:"omitempty,numeric,max=16"`
    RoleID   uint   `json:"role_id"   binding:"omitempty,gt=0"`
    Status   *int8  `json:"status"    binding:"omitempty,oneof=0 1"`
}

type ChangePasswordReq struct {
    OldPassword string `json:"old_password" binding:"required,min=6,max=64"`
    NewPassword string `json:"new_password" binding:"required,min=6,max=64"`
}

// ── 响应 ── (嵌入在 Response.Data 中)
type UserInfo struct {
    ID        uint      `json:"id"`
    Username  string    `json:"username"`
    RealName  string    `json:"real_name"`
    Email     string    `json:"email"`
    Phone     string    `json:"phone"`
    RoleID    uint      `json:"role_id"`
    Role      *RoleInfo `json:"role,omitempty"`
    Status    int8      `json:"status"`
    CreatedAt time.Time `json:"created_at"`
    UpdatedAt time.Time `json:"updated_at"`
}

type RoleInfo struct {
    ID          uint   `json:"id"`
    RoleName    string `json:"role_name"`
    Description string `json:"description"`
}
```

### 6.3 设备模块

```go
// ── 请求 ──
type CreateEquipReq struct {
    Name        string `json:"name"        binding:"required,min=1,max=128"`

type EquipmentInfo struct {
    ID             uint      `json:"id"`
    Name           string    `json:"name"`
    Model          string    `json:"model"`

    CreatedAt      time.Time `json:"created_at"`
    UpdatedAt      time.Time `json:"updated_at"`
}
```

### 6.4 借阅模块

```go
// ── 请求 ──
type ApplyBorrowReq struct {
    EquipmentID uint   `json:"equipment_id" binding:"required,gt=0"`
    Quantity    uint   `json:"quantity"     binding:"required,gt=0"`
    ApplyNote   string `json:"apply_note"   binding:"max=256"`
}

type ApproveBorrowReq struct {
    Approve     bool   `json:"approve"`
    ApproveNote string `json:"approve_note" binding:"max=256"`
}

// ── 响应 ──
type BorrowRecordInfo struct {
    ID          uint            `json:"id"`
    UserID      uint            `json:"user_id"`
    EquipmentID uint            `json:"equipment_id"`
    Quantity    uint            `json:"quantity"`
    Status      string          `json:"status"`
    ApplyNote   string          `json:"apply_note"`
    ApproveNote string          `json:"approve_note"`
    ApproverID  uint            `json:"approver_id"`
    ApplyAt     time.Time       `json:"apply_at"`
    ApproveAt   *time.Time      `json:"approve_at"`
    ReturnAt    *time.Time      `json:"return_at"`
    // 关联（仅在列表/详情接口中展开）
    User      *UserBrief      `json:"user,omitempty"`
    Equipment *EquipmentBrief `json:"equipment,omitempty"`
    Approver  *UserBrief      `json:"approver,omitempty"`
}

type UserBrief struct {
    ID       uint   `json:"id"`
    Username string `json:"username"`
    RealName string `json:"real_name"`
}

type EquipmentBrief struct {
    ID    uint   `json:"id"`
    Name  string `json:"name"`
    Model string `json:"model"`
}
```
