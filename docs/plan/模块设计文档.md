## 编排原则 (v2.0 vs v1.0)

| v1.0 (旧编排) | v2.0 (新编排) | 合并理由 |
|---------------|---------------|---------|
| 01-认证鉴权 + 02-用户组织 + 05-权限审计 | **01-身份与访问管理** | 共享 `sys_users` + `sys_roles` + `casbin_rule` 三张表，Auth→Casbin 中间件在同一请求链上紧耦合 |
| 03-设备资产 | **02-设备资产管理** | 保持不变，标注了与借阅模块的跨表事务耦合 |
| 04-借阅工单 | **03-借阅工单管理** | 保持不变，标注了与设备模块的跨表事务耦合 |
| 06-基础公共 | **04-系统基础设施** | 从业务模块序列移出，定位为 Shared Kernel |
